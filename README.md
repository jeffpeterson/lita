# Lita Lang

## Debugger

Start the debugger with: `make db/<example>` E.g. `make db/basics`

### lldb cheatsheet

| Alias | Command | Help                             |
| ----- | ------- | -------------------------------- |
| `r`   | `run`   | Run the program.                 |
| `up`  |         | Move up a frame.                 |
| `v`   | `var`   | Show variables in current frame. |
| `p`   | `print` | Evaluate an expression.          |
| `q`   | `quit`  | Quit the debugger.               |
